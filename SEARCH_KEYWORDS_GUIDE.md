# מדריך שיפור חיפוש - Search Keywords

## הבעיה
משתמשים מחפשים "כסא" אבל המוצר נקרא "כיסא" (או להפך) ולא מוצאים אותו.

## הפתרון המומלץ ✅

### 1. הוספת שדה `search_keywords` למוצרים

**יתרונות:**
- ✅ SEO מצוין - מילות מפתח נוספות
- ✅ וריאציות כתיב (כסא/כיסא, שולחן/שלחן)
- ✅ מילים נרדפות (כסא = chair)
- ✅ פשוט לנהל ולעדכן
- ✅ עובד מיד ללא אלגוריתמים מורכבים

### 2. הרצת הסקריפט

```bash
# התחבר ל-Supabase SQL Editor והרץ:
cat add-search-keywords.sql
```

או דרך Supabase Dashboard → SQL Editor → הדבק את הקוד

### 3. איך להוסיף keywords למוצרים חדשים

כשמוסיפים מוצר חדש, מלא את השדה `search_keywords`:

**דוגמאות:**

| שם המוצר | search_keywords |
|----------|----------------|
| כיסא מעוצב | כסא, כיסא, כסאות, כיסאות, chair, seat |
| שולחן עבודה | שולחן, שלחן, שולחנות, table, desk |
| נורת LED | נורה, נורות, נורת, bulb, light, led |
| ספה תלת מושבית | ספה, ספות, couch, sofa, ספה תלת |

### 4. עדכון המוצרים הקיימים

אופציה א': דרך Supabase Dashboard
```sql
UPDATE products 
SET search_keywords = 'כסא, כיסא, כסאות, כיסאות, chair'
WHERE id = 'המזהה-של-המוצר';
```

אופציה ב': דרך Webhook/Import
- עדכן את ה-webhook שמייבא מוצרים מ-CRM
- הוסף לוגיקה שמייצרת keywords אוטומטית

### 5. יצירת keywords אוטומטית (אופציונלי)

```javascript
function generateSearchKeywords(productName, category) {
  const keywords = [productName];
  
  // הוסף וריאציות נפוצות
  const variations = {
    'כיסא': ['כסא', 'כיסא', 'כסאות', 'כיסאות'],
    'שולחן': ['שולחן', 'שלחן', 'שולחנות'],
    'נורה': ['נורה', 'נורות', 'נורת'],
    'מנורה': ['מנורה', 'מנורות', 'מנורת'],
  };
  
  // חפש התאמות והוסף וריאציות
  for (const [key, values] of Object.entries(variations)) {
    if (productName.includes(key)) {
      keywords.push(...values);
    }
  }
  
  return keywords.join(', ');
}
```

## תוצאות צפויות

### לפני:
- חיפוש "כסא" → 0 תוצאות (אם המוצר נקרא "כיסא")

### אחרי:
- חיפוש "כסא" → מוצא "כיסא מעוצב", "כסאות בר", וכו'
- חיפוש "chair" → מוצא את כל הכיסאות (תמיכה באנגלית)
- חיפוש "כיסא" → מוצא גם "כסא"

## יתרונות נוספים ל-SEO

1. **Meta Keywords** - מילות המפתח משפרות את הדירוג
2. **Rich Snippets** - Google מבין טוב יותר את המוצר
3. **Long-tail Keywords** - תפיסת חיפושים ספציפיים
4. **Multi-language** - תמיכה בעברית ואנגלית

## תחזוקה שוטפת

### מומלץ:
1. בדוק אילו מילים משתמשים מחפשים (Google Analytics)
2. עדכן keywords על בסיס חיפושים ללא תוצאות
3. הוסף מילים נרדפות מקטגוריה/תת-קטגוריה
4. כלול שמות מותגים (אם רלוונטי)

### דוגמה מתקדמת:
```
שם: כיסא גיימינג ארגונומי
keywords: כסא, כיסא, כסאות, גיימינג, gaming, chair, ארגונומי, ergonomic, כסא גיימר, gaming chair, כיסא משחקים
```
